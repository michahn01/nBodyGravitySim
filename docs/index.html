Controls:

Press z to zoom in on mouse cursor.
Press LSHIFT z to zoom out.
Arrow keys to move camera.

Keep your browser in full-screen for best experience.

Spawning Particles:

Use SPAWN PARTICLES drop-down button to spawn particles. Select number and mass of each particle.

Particles can be spawned at any time, whether the simulation is paused, being run, or not yet started.

Using an Example Config:

STAR SYSTEM Generic star system with one star and two planets that orbit it.

BINARY STAR SYSTEM Star system with two stars of equal mass orbiting each other.

UNSTABLE 3 BODY Three bodies of equal mass. Chaotic and highly sensitive to the point where using floating-point vs double precision yields different results. An illustration of the Three-Body Problem.

EULER'S 3 BODY A stable three-body configuration discovered by Leonhard Euler.
